# 🚀 Star Defender

웹 브라우저에서 플레이하는 슈팅 게임!

## 🎮 [지금 플레이하기](https://jangws1030.github.io/star-defender/)

---

## ⌨️ 조작법

| 키 | 동작 |
|:---:|:---|
| ← → ↑ ↓ | 이동 |
| SPACE | 발사 |
| Z | 스킬 (게이지 100%) |
| R | 재시작 |
| Q | 종료 |

---

## 🛡️ 파워업

| 아이템 | 효과 |
|:---:|:---|
| ⚡ POWER | 발사 파워 증가 |
| 🔥 SPEED | 발사 속도 증가 |
| 💨 MOVE | 이동 속도 증가 |
| 🛡️ SHIELD | 10초간 보호막 |
| 💚 GAUGE | 스킬 게이지 충전 |
| 💥 CLEAR | 화면 적 전멸 |

---

## ✨ 특수 스킬 이펙트

- ⭐ **StarBurst** - 별 파티클 폭발
- 🔥 **RocketFlame** - 엔진 화염 트레일
- 🛡️ **MultiLayerShield** - 3층 회전 쉴드
- 💫 **ShockWave** - 충격파 이펙트

---

## 🛠️ 기술 스택

- **Python** + **Pygame-CE**
- **Pygbag** (웹 배포)
- **GitHub Pages** (호스팅)
